Overview

This project analyzes healthcare spending patterns to uncover insights about costs and utilization. Using SQL and data visualization, it explores which factors contribute most to healthcare expenses and highlights potential areas for cost optimization.

Features
Data cleaning and preparation using SQL
Analysis of costs across procedures, providers, and patient demographics
Visual dashboards (pie charts, bar charts) to display trends
Identification of high-cost areas and opportunities for savings

Key Insights
Some procedures and patient groups drive most of the healthcare costs
Preventive care and early interventions can reduce overall spending
Visualizations make patterns easy to understand and communicate

Tools Used
SQL (for data extraction and querying)
Excel / Tableau / Python (for visualization, optional depending on your workflow)
